# Vantix AI Builders

Complete full-stack prototype with Credits system, Import/Deployment Center, Auto-Preview, Secure Admin Command Center, Multi-Model AI Gateway, Dev Tools Suite, AI Pro Skills Registry, Intelligence Engine, Project Memory, and private billing controls.

## Run locally
```bash
npm install
cp .env.example .env
npm run dev
```

## URLs
- Builder: http://localhost:3000
- Admin: http://localhost:3000/admin
- API Health: http://localhost:4000/api/health
- Credits: http://localhost:4000/api/credits/demo-user
- Imports: http://localhost:4000/api/imports
- Deploy Providers: http://localhost:4000/api/deploy/providers
